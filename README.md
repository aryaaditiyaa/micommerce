## About Micommerce

Aplikasi mini-ecommerce yang dibangun menggunakan techstack sebagai berikut:
- Tailwind CSS (Fully Mobile Responsive)
- Tailwind UI
- Laravel 8
- HeroIcons
- MySQL
- maatwebsite/excel
